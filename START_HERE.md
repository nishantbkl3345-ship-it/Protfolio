## 🔧 Common Customizations

### Change Colors
1. Open `app/globals.css`
2. Find `#3b82f6` (blue) and `#9333ea` (purple)
3. Replace with your preferred colors

### Update Projects
1. Open `components/Projects.tsx`
2. Edit the `featuredProjects` array
3. Add/remove projects as needed

### Modify Skills
1. Open `components/Skills.tsx`
2. Edit the `skillCategories` array

### Change Personal Info
1. `components/Hero.tsx` - Name and tagline
2. `components/About.tsx` - Bio
3. `components/Contact.tsx` - Contact details

## 🚢 Deploy Your Portfolio

### Option 1: Vercel (Recommended - 5 minutes)
1. Push your code to GitHub
2. Go to [vercel.com](https://vercel.com)
3. Click "New Project"
4. Import your repository
5. Click "Deploy"

Done! Your site will be live with a URL like: `your-name.vercel.app`

### Option 2: Netlify
1. Push to GitHub
2. Go to [netlify.com](https://netlify.com)
3. Import your repository
4. Deploy

See [DEPLOYMENT.md](./DEPLOYMENT.md) for detailed instructions.
